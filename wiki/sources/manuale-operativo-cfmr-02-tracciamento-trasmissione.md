**Type**: Operational Manual

## Summary

Manual detailing the IT systems and procedures for tracking and transmitting CFMR risk evaluations through operational systems. Covers documentation of decision-making processes using Windows Action Recorder, management of ARPA meteorological bulletins, color code approval workflows, and updates to alert applications and databases.

## Key claims

- Decision-making process is documented using Windows PSR tool with screenshot sequences
- Meteorological models (Ghost, FEWS, Sinergie) are systematically reviewed and recorded
- ARPA vigilance bulletins are archived daily and linked to issued alert codes
- Color code proposals require approval before transmission to Regional Operations Room
- AllertaLOM application and DEWETRA-WebAlert system must be updated with approved alerts
- Complete audit trail maintained through registration system
