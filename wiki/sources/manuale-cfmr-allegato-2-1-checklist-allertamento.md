# Allegato 2.1: Check List - Allertamento CFMR

**Source**: `raw/2023.10.18_Decreto n. 15960 - Approvazione PROCEDURE CFMR/Allegato 2.1 - Check list_Allertamento CFMR.pdf`
**Ingested**: 2026-05-12
**Type**: Quality Assurance Checklist

## Purpose

Verification checklist for [[centro-funzionale-monitoraggio-rischi]] operators conducting technical evaluation activities and alert issuance. Ensures all evaluation steps completed before color code proposal.

## Scope

Applied by CFMR operators throughout daily risk assessment and alert generation workflow as referenced in "TRACCIAMENTO E TRASMISSIONE DELLA VALUTAZIONE" manual.

## Key Areas

Systematic verification of:
- Meteorological model consultation (Ghost, FEWS, Sinergie)
- ARPA vigilance bulletin reception and analysis
- Risk assessment completion per homogeneous zone
- Color code determination process
- Approval and transmission procedures
- System updates (AllertaLOM, GESTCOM, DEWETRA)
