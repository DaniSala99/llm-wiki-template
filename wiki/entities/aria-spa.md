# ARIA Spa

**Type**: Technology Company / Service Provider

## Overview

ARIA Spa is the technical support provider and software developer for civil protection operational systems used by Regione Lombardia. Develops and maintains GESTCOM application and provides IT infrastructure support for CFMR operations.

## Related systems

- GESTCOM - Application for alert management and system updates
- AllertaLOM - Alert dissemination application
- DEWETRA-WebAlert - Public alert system
